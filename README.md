
# Schweizerischer KMU Verein Website

This is the website for the Schweizerischer KMU Verein, a Swiss association supporting small and medium businesses across Switzerland.

## Technologies Used

- React
- TypeScript
- Tailwind CSS
- shadcn/ui

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Deployment

The site is automatically deployed through a CI/CD pipeline.
